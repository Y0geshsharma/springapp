## Spring Boot WebSocket Chat Appplication


## Requirements

1. Java - 1.8.x

2. Maven - 3.x.x

## Steps to Setup

**1. Clone the application**

```bash
git clone https://github.com/y0geshsharma/springapp.git
```

**2. Build and run the app using maven**

```bash
cd springapp
mvn package
java -jar target/websocket-demo-0.0.1-SNAPSHOT.jar
```
